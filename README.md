## 🔨 Setting up locally

Create virtualenv and install dependencies

```sh
make setup
```

## ⚡️ Running the application

Running locally:

```sh
make run
```

Running in a container:

```sh
make docker-run
```

## ✨ Linters and Formatters

Check for linting rule violations:

```sh
make check
```

Auto-fix linting violations:

```sh
make fix
```

## 🤸‍♀️ Getting Help

```sh
make

# OR

make help
```
